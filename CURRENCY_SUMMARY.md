## Проверка работы

После применения миграции проверьте:

1. **Список валют:**
   ```bash
   GET /api/v1/wallet/tokens
   ```
   Должно вернуть 25 валют

2. **Лимиты ставок:**
   ```bash
   GET /api/v1/wallet/bet-limits
   ```
   Должно вернуть лимиты для всех валют

3. **Пополнение:**
   - Откройте страницу пополнения
   - Должен быть выбор из всех 25 валют
   - Можно выбрать любую валюту для пополнения

4. **Игры:**
   - Попробуйте сделать ставку в любой игре
   - Максимальная ставка должна соответствовать выбранной валюте
