# Настройка валют и максимальных ставок

## ⚠️ ВАЖНО: Применение миграции

Перед использованием системы валют необходимо применить миграцию базы данных!

См. файл `CURRENCY_MIGRATION_GUIDE.md` для подробных инструкций.

## Поддерживаемые валюты

### Стейблкоины (1:1 к USD)
- **USDT** - TRC-20, ERC-20, BEP-20, SOL, OP, ARB, TON
- **USDC** - ERC-20, BEP-20, SOL, OP, BASE, ARB
- **USDD** - TRC-20
- **TUSD** - ERC-20, BEP-20

### Основные криптовалюты
- **BTC** - Bitcoin (BTC сеть)
- **ETH** - Ethereum (ERC-20, OP)
- **TRX** - TRON (TRC-20)
- **LTC** - Litecoin (LTC сеть)
- **BNB** - Binance Coin (BEP-20)
- **SOL** - Solana (SOL сеть)
- **TON** - The Open Network (TON сеть)
- **SHIB** - Shiba Inu (ERC-20)

## API Endpoints

### Получить список валют
```
GET /api/v1/wallet/tokens
GET /api/v1/deposit/currencies
```

### Получить лимиты ставок для всех валют
```
GET /api/v1/wallet/bet-limits
```

### Получить лимиты ставок для конкретной валюты
```
GET /api/v1/plinko/config?tokenId=1
```

## Игры

Все игры поддерживают работу с любыми валютами:
- **Crash** - валидация ставок по лимитам валюты
- **Minesweeper** - валидация ставок по лимитам валюты
- **Plinko** - валидация ставок по лимитам валюты

## Автоматическая синхронизация

При запуске сервера автоматически синхронизируются все валюты из CryptoBot и CryptoCloud в базу данных.

## Обновление курсов валют

Курсы валют хранятся в `currencySyncService.js` в объекте `CURRENCY_RATES`. 
Для обновления курсов измените значения в этом объекте.
